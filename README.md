## Install

```bash
git clone https://github.com/SashaSarzh/InstagramDirectBomber.git
```

### Features

- Instagram direct bombing

## Before Running `bomber.py`


* Windows and Linux:

```bash
pip3 install -r requirements.txt

```

## Requirements

*  Python 3.x

## Run

* Windows :

```bash
python bomber.py
```
* Linux :

```bash
python3 bomber.py
```
